# Página de Login
Este projeto é uma página de login funcional desenvolvida com as seguintes tecnologias:  
- HTML  
- CSS  
- JavaScript  
- React  

## Objetivo
Criar uma aplicação simples, responsiva e funcional que permita o login de usuários.  

## Funcionalidades
- Validação de campos (e-mail e senha).  
- Design responsivo.  

## Como executar
1. Clone o repositório:  
https://github.com/pan-ti/pagina-login.git

2. Abra o arquivo `index.html` no navegador.
